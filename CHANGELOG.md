# Changelog

Format based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.2.0] — 2026-02-28

### Phase 2 — Core Features

#### Added
- **Live GPS Tracking** — Animated map with real-time van movement, ETA countdown, status pipeline
- **Customer Tracking Links** — Shareable URLs for end customers (e.g. `loadlink.co/track/LL-0847`)
- **Digital Proof of Delivery** — Photo capture + signature pad + GPS stamp + timestamp
- **Rating & Review System** — 5-star ratings, written reviews, rating breakdown charts
- **Payments Dashboard** — Wallet balance, bank withdrawal, instant payout, transaction history
- **Auto Invoicing** — Generated on POD submission, status tracking, download, payment reminders
- **Push Notifications** — Bell icon with unread count, slide-out panel, multiple notification types
- **Weekly Earnings Summary** — Earned / Fees / Jobs breakdown

#### Changed
- Business booking flow expanded from 4 to 5 steps (added Tracking + POD/Rate)
- Driver dashboard expanded from 4 to 8 tabs
- Navigation now includes notification bell

---

## [0.1.0] — 2026-02-28

### Phase 1 — MVP

#### Added
- **Landing Page** — Role selection (Business / Driver) with feature highlights
- **Business Booking Flow** — 4-step wizard (Details → Vehicle → Price → Confirmation)
- **3 Booking Modes** — Quote Request, Fixed Price (Uber-style), Driver Availability
- **Driver Dashboard** — Load board, active jobs, availability posting, driver browse
- **Load Board** — Filtering by type (All / Fixed / Quote), instant accept, quote submission
- **Vehicle Selection** — 6 categories (Small Van → 7.5t Truck) with payload info
- **Service Tiers** — Economy / Standard / Express with dynamic pricing
- **Driver Availability** — Post location, heading, vehicle, range, hours, rate
- **Special Requirements** — Tail lift, fragile, 2-man, temperature controlled
- **Rate Guidance** — Average rate per mile shown when quoting
- **Mobile Responsive** — Works on all screen sizes
